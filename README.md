## 🚀 How to Run

1. Clone the repository  
   ```bash
   git clone https://github.com/Gaurikadu13/JFS_Learning.git
   ```

2. Go to project folder  
   ```bash
   cd JFS_Learning
   ```

3. Navigate to core_java folder  
   ```bash
   cd core_java
   ```

4. Compile the program  
   ```bash
   javac FileName.java
   ```

5. Run the program  
   ```bash
   java FileName
   ```
